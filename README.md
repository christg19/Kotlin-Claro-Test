# Contactos App

Prueba técnica Android — gestión de contactos.

## Stack

- Kotlin + Jetpack Compose
- MVVM + Clean Architecture
- Hilt, Room, Retrofit, Coroutines + Flow, Coil

## Funcionalidades

- Listado con búsqueda por nombre, apellido y teléfono
- Selección y borrado múltiple
- Creación de contactos con foto aleatoria (picsum.photos)
- Persistencia local con Room

## Tests

Tests unitarios en ViewModels con MockK y kotlinx-coroutines-test.

## Ejecutar

Abrir en Android Studio y dar Run. No requiere API key.
